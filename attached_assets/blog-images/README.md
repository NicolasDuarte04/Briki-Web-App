# Blog Images

Professional curated images for Briki blog articles. These images are optimized for web use and match Briki's visual brand.

## Image Categories

- **travel-insurance.jpg** - Modern travel theme for travel insurance articles
- **auto-insurance.jpg** - Vehicle protection themes for auto insurance content  
- **pet-insurance.jpg** - Pet care and protection imagery
- **health-insurance.jpg** - Medical and wellness themes
- **insurance-tips.jpg** - General advice and money-saving themes

## Usage

Import images in React components:
```jsx
import travelImage from "@assets/blog-images/travel-insurance.jpg";
```
